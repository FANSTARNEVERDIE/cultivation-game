# 修仙模拟器

一款纯本地运行的文字修仙模拟器，体验从凡人到仙人的修仙之路。

## 🎮 游戏特色

- 🧘 修炼系统 - 14个境界，从练气期到大罗金仙
- ⚔️ 战斗系统 - 暴击、闪避、法术、灵兽助战
- 📖 剧情系统 - 25章主线 + 30个随机事件
- 💕 社交系统 - 道侣、灵兽、NPC互动
- 🏔️ 副本系统 - 随机副本、深层探索
- 🧪 炼制系统 - 炼丹、炼器、布阵、画符

## 🚀 如何玩

1. 打开 `index.html` 首页
2. 点击"开始修仙"进入游戏
3. 输入邀请码开始游戏
4. 从凡人开始，踏上修仙之路！
